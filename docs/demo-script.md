# Scripted Demo — ~3 minutes (Chunk 7.5)

A reliable, repeatable flow you can run on command. The story is one line:
**the body comes back faster than the head, and Comeback shows the gap.**

## Before you start

- **Load sample data** so the trends exist: Settings (⋯ on the Dashboard) →
  *Load sample data* (or onboard via *Explore with sample data*). A fresh user
  has no trend — never demo the gap on an empty account.
- **Record a screen-capture backup** of this exact flow. Live camera demos fail
  at the worst moment; have the recording ready to fall back to.
- Good light, room to stand side-on, phone held level.

## The path

1. **Dashboard (15s).** Open on the Dashboard. "This is a comeback in one view —
   physical readiness up top, and the **Readiness Gap**: body vs. head." Point at
   the gap meter.

2. **Measure (45s).** Tap **Measure**. Stand side-on, whole body in frame — the
   guide turns green when the shot is good (that gate is what keeps the number
   honest). Run the guided rep: *straighten*, then *bend*. Show the live
   knee-angle number tracking the leg. Save.
   - Credibility line: *"validated to within 3–5° of a goniometer."*

3. **Confidence (45s).** Tap **Confidence → Start a check**. Answer a few of the
   ACL-RSI items (one per screen). Show the result: overall score **plus the
   three subscales** — and that **risk appraisal lags**, which is expected. "No
   other consumer tool measures this."

4. **Trend → the Readiness Gap (45s).** Tap **Trend**. Green line = the knee's
   physical recovery. White line = confidence. "The body climbs; the head trails.
   The space between them is the thing clinics miss — and the thing we make
   visible."

5. **Ladder (30s).** Tap **Ladder**. "Fear of cutting is the last barrier, so we
   grade the exposure — walking-pace changes of direction up to cutting in a
   game." Mark the next rung complete; show the climb advance.

6. **Close (10s).** Back to Dashboard. "Range within a few degrees of a
   goniometer, confidence on a validated scale, and the gap between them —
   companion, not clearance."

## Reset between runs

Settings → *Reset all data* returns to the start; then *Load sample data* to run
it again cleanly.
